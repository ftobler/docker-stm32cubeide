
# STM32CubeIDE in Docker

[![build_docker](https://github.com/ftobler/docker-stm32cubeide/actions/workflows/build_docker.yml/badge.svg)](https://github.com/ftobler/docker-stm32cubeide/actions/workflows/build_docker.yml)

This project provides a Docker image with [STM32CubeIDE](https://www.st.com/en/development-tools/stm32cubeide.html) installed, based on Ubuntu and Node.js.

[ftobler/stm32cubeide](https://hub.docker.com/r/flotobler/stm32cubeide)


## Features

- Ubuntu base image
- STM32CubeIDE pre-installed
- Node.js included
