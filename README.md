
# STM32CubeIDE in Docker

This project provides a Docker image with [STM32CubeIDE](https://www.st.com/en/development-tools/stm32cubeide.html) installed, based on Ubuntu and Node.js.

## Features

- Ubuntu base image
- STM32CubeIDE pre-installed
- Node.js included
